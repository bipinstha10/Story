Here, the file called 'chapter1.txt' is inside Story folder and the Story is the working directory. To track the changes made in the file chapter1.txt we need to add the file to the "Staging Area". We use the command "git status" to see what are inside the staging area. Now to add the file chapter1.txt to the staging area means to track the changes on the file we use the command "git add chapter1.txt"
